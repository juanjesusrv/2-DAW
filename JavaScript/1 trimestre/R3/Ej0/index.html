Validador de correo electronico

Ejercicio 1. Realiza una aplicación web dónde se solicite por pantalla, mediante una ventana emergente, al usuario una cadena de caracteres.

Dicha cadena de caracteres corresponde a un formato válido de email.

  • Nombre de usuario, compuesto por letras mayúsculas o minúsculas, o números, o puntos, o guión bajo o alto
  • Una arroba
  • Nombre de servidor, que puede contener los mismos caracteres que el nombre de usuario excepto el punto
  • El dominio, que podrá estar formado por uno o dos grupos de 2 a 4 letras separados por puntos. Por ejemplo, consideramos válido .es, .com, o .com.es.
Los formatos válidos son del tipo:
  1. carmelo@gmail.com
  2. carmelo123@gmail.com
  3. Carmelo_Jaen@gmail.com
  4. carmelo-jaen@gmail.com
  5. Carmelo.jaen@gmail.com
  6. carmelo@gmail.com.es
  7. carmelo.jaen-diaz@gmail.site
